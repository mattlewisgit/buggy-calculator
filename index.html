User Story

Calculator available at: https://mattlewisgit.github.io/buggy-calculator/kingfisher-calculator.html

Title: Web Calculator

Description: As a Kingfisher user, I want to perform basic arithmetic operations (addition, subtraction, multiplication, division) using a web-based calculator so that I can quickly calculate numbers in my browser.

Acceptance Criteria

  • Adding numbers together works as expected.
  • Dividing numbers together works as expected.
  • Multiplying numbers together works as expected.
  • Subtracting numbers together works as expected.
  • No unexpected behaviour is observed from any other buttons.
  • The calculator responds within milliseconds with the result of the calculation.
